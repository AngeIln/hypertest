# Product Spec – AI Code Studio

## Vision & objectifs (MVP)
- Génération + explication pour Python, JavaScript, Java, C#, SQL.
- Éditeur web et exécution sandbox restreinte Python/JS.
- Historique des sessions + versioning simple.
- Authentification utilisateur.
- Lancement de tests unitaires depuis l'interface.

## Public cible
- Étudiants débutants.
- Développeurs professionnels.
- Enseignants.

## Roadmap
### MVP
- Auth email/password + base OAuth.
- Éditeur web + backend IA.
- Exécution limitée Python/JS.
- Dashboard, historique, profil.

### v2
- Collaboration temps réel.
- Génération + exécution de tests unitaires.
- GitHub push/PR automation.
- Mode code review (lint + IA).
- Paramétrage des modèles IA.

### v3
- Pair programming vocal/vidéo.
- Extension VSCode.
- Marketplace snippets.
- Fine-tuning privé.
- Analyse de complexité automatique.
